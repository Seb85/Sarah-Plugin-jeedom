Le plugin JEEDOM permet de controler votre box JEEDOM en utilisant les commandes vocales de S.A.R.A.H.

Les commandes vocales sont définies par les interactions realisees et synchronisees depuis votre box JEEDOM :

  • Sarah ouvre le volet de la cuisine
  • Sarah allume le home cinema
  • Sarah quelle est la temperature du salon

Le fichier des interactions jeedom.xml est généré automatiquement après synchronisation dans JEEDOM
